# lyzoria_studios

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Open the app:

```text
http://localhost:3000
```

## Build and run production

```bash
npm run build
npm run start
```
